Web Bluetooth needs Chrome or Edge over https or http://localhost (a secure context) — serve this folder, e.g. python3 -m http.server from the repo root, then open http://localhost:8000/web/. The sensor must expose the Heart Rate service with RR intervals (e.g. Polar H10). Metrics stabilise as data accumulates (frequency-domain needs ~1–5 min per block). Reverse-engineering / research tool — not a medical device.
